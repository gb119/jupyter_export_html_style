## [0.1.0] - 2026-01-11

### Added
- Initial release of jupyter-export-html-style
- StylePreprocessor for handling cell style metadata
- StyledHTMLExporter for HTML export with custom styles
- Comprehensive documentation with Sphinx
- Build configurations for both Python wheels and conda packages
- GitHub Actions workflows for CI/CD
- Support for both dictionary and CSS string style formats
- Entry points for nbconvert integration
- Full test suite structure

### Documentation
- Installation guide
- Usage guide with examples
- API reference
- Contributing guidelines
- Sphinx-based documentation structure

### Build System
- pyproject.toml with modern build configuration
- Conda recipe for conda-forge distribution
- GitHub Actions for automated building and testing
- Support for Python 3.11+
